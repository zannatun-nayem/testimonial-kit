Getting started

Welcome

Hello there! Thanks for buying Marturie. Marturie is a Custom testimonial with solid design especially crafted to help you built any kind of websites with huge variety of options.

This UI KIT is suitable for all types of Templates. This Marturie built using Twiiter's Bootstrap 5.0 Framework, Google Fonts, Font Awesome 5.15.3 It is highly recommended that you are familiar in a way with at least HTML5 / CSS3/SASS .

So let's jump directly to it, and start everything from scratch.

Downloading Marturie

Once you download the zip file, unzip it. You should have a folder with the following folder & files by now.

Setting your fonts

To use google fonts, head over to Google's Fonts and add whichever font you want to your collection. Once you finished, follow the instructions and place the generated link in custom.css file.
The link should look like this : 

<link
								href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,300;0,400;0,700;1,300&display=swap"rel="stylesheet" />
Next, just replace the font family, with the font family you just copied. Like: 
 body {font-family: "Roboto", sans-serif;}
